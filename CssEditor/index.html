üìö Docs & Tips

  • Selectors: Use .sample for the preview, or body, .editor-panel, etc. for the website.
  • Common Properties:
    • color, background, font-size, padding, margin
    • border, border-radius, box-shadow, transition
    • display: flex / grid for layout
  • Pseudo-classes: :hover, :active, :focus
  • Variables: Use var(--accent) for site accent color.
  • Example: 
    .sample {
  color: #7b2ff2;
  background: #e0e7ef;
  border-radius: 10px;
  box-shadow: 0 3px 10px #7b2ff218;
}
